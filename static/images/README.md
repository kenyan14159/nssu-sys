# 公印画像について

領収書に表示する公印（角印）画像をここに配置してください。

## ファイル名
`seal.png`

## 要件
- 形式: PNG（透過背景推奨）
- 推奨サイズ: 300x300px 以上
- 背景: 透過処理済み

## 作成手順
1. 日体大陸上部の角印を白い紙に押印
2. スマートフォンで正面から撮影
3. 背景透過処理を行う（以下のツールが便利です）
   - remove.bg（無料）
   - Canva（無料）
   - Adobe Express（無料）
4. 透過PNG形式で保存
5. このフォルダに `seal.png` として配置

## 注意事項
- 公印画像がない場合、領収書には代替の円形テキストが表示されます
- 正式な書類として認めてもらうためには、実際の公印画像の使用を推奨します
- 画像ファイルは Git で管理しないことを推奨します（.gitignore に追加済み）
